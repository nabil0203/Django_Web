
# Project Details


## Overview
- Develop a To-Do Application using Django, allowing users to manage their tasks efficiently with authentication and filtering features.

## Requirements:
1️⃣Task Management
- Users can add new tasks with details such as title, description, due date, due
time.

2️⃣Task Viewing & Filtering
- Users can view all tasks, including both completed and remaining tasks.
- Users can filter tasks by status:
- Completed Tasks
- Pending/Remaining Tasks
