# Render Free Hosting


## 1. Run command in the main project which has app in it
```bash
python -m pip freeze > requirements.txt
```
- a `requirements.txt` file will be created
- this will show all the packages
- remove the versions




## 2. Visit [Render](https://render.com/)
- Click `New`
- Select `Web Service`



## 3. Fill-up Commands
- **Build Command:**
```
$ pip install -r requirements.txt
```
- **Start Command:**
```
$ python manage.py runserver 0.0.0.0:80
```



## 4. `settings.py`
```py

ALLOWED_HOSTS = [
    'taskify-cq2h.onrender.com',
    'localhost',
    '127.0.0.1',
]

CSRF_TRUSTED_ORIGINS = [
    'https://taskify-cq2h.onrender.com',
    'https://*.onrender.com',
]
```
