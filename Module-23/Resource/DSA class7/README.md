# Class 7

## Topics to Discuss:

- Graph representations: adjacency list & matrix
- DFS and BFS

- Build simple graph using dictionary
- Perform DFS and BFS traversal using queue and stack

## Homework:
- Represent undirected graph in Python
- Solve one problem using BFS and another using DFS
