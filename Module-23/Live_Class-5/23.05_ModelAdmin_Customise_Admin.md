### 1ï¸âƒ£ `admin.py`
- We want to change "Category" model
```py
@admin.register(models.Category)
class CategoryAdmin(admin.ModelAdmin):
    list_display = ['name', 'slug']
    prepopulated_fields = {'slug' : ('name',)}
```
- **ModelAdmin** is used to customize 
- `list_display` is used to show the entities we wanna see in admin panel
    - In this example:
        - Name & Slug wil be shown
- `prepopulated_fields` is used to fill a section automatically
    - - In this example:
        - While filling Name
        - Slug wil be auto fill
