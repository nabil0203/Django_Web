# üåê `Global` Templates (Common for the whole project)

## 1Ô∏è‚É£ Create `templates` folder

- Go to the main project directory (the one that contains manage.py).
- Create a folder named `templates`.

Example structure:
```bash

    ‚îÇ
    ‚îú‚îÄ‚îÄ my_project                          # inner project
    ‚îú‚îÄ‚îÄ templates                           # üëà new folder for global templates
```

------------------------------------

## 2Ô∏è‚É£ Tell Django about this folder

- We have not said anywhere that, a folder named `templates` has been created.
- To do that:

    - Open `settings.py` of inner project folder.
    - Scroll down and find the `TEMPLATES` section.
    - In `'DIRS'`: [ ], add the path to your templates folder.

```bash
TEMPLATES = [
    {
        'BACKEND': 'django.template.backends.django.DjangoTemplates',
        
        'DIRS': ["templates"],                                      # üëà added line

        'APP_DIRS': True,
        'OPTIONS': {
            'context_processors': [
                'django.template.context_processors.debug',
                'django.template.context_processors.request',
                'django.contrib.auth.context_processors.auth',
                'django.contrib.messages.context_processors.messages',
            ],
        },
    },
]
```

## 3Ô∏è‚É£ Create an HTML file
- Inside the ``templates/`` folder, make a file `index.html`.
- Example:
```html

    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>

    <h1>This is Home page using HTML</h1>


</body>

```

## 4Ô∏è‚É£ Render the template from a view
- In your app‚Äôs `views.py`:
```python
from django.shortcuts import render

def home(request):
    return render(request, 'index.html')                # Django looks in global templates

```

------------------------------------


## 5Ô∏è‚É£ Connect the view to a URL
- In your app‚Äôs `urls.py`:

```python
from django.urls import path
from . import views

urlpatterns = [
    path('', views.home, name='home'),
]

```
---------------------------------

‚úÖ Now when you run the server and visit http://127.0.0.1:8000/, Django will show your index.html from the global templates folder.
