# Block Tag


## `{% block %}` 
- The block tag is used to for `overriding` specific parts of a **parent template**.


---------------------------------------


Syntax :-

- {% block blockname %} ........ {% endblock %}

- {% block blockname %} ........ {% endblock blockname %}


--------------------------------


Example :-

- {% block title %} ......... {% endblock %}

- {% block content %} ......... {% endblock content %}
