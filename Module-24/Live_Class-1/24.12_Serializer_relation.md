    name = models.CharField(max_length=100)
    bio = models.TextField()

    def __str__(self):
        return self.name


class Book(models.Model):
    author = models.ForeignKey(Author, on_delete=models.CASCADE, related_name='books')
    title = models.CharField(max_length=100)
    published_date = models.DateField()


    def __str__(self):
        return self.title

## âœ… Why this works

- Author â†” Book is a One-to-Many relationship
    - One author can write multiple books.

- related_name='books'
    - Allows Django/DRF to access all books of an author using --> `author.books`


## ğŸ“Œ`StringRelatedField(many=True)`:
- Uses the `__str__()` method of the related Book model
- Returns a read-only list of string values
- Ideal for human-readable output in APIs
