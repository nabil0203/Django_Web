# Crispy Form in Django
- Designs the Form in a stylish way **without CSS**

-------------------------------------------------------------------


# Visit
- for step by step setup follow the GeekForGeeks resource:
```
https://www.geeksforgeeks.org/python/styling-django-forms-with-django-crispy-forms/
```

-----------------------------------------------------------------------


# 1️⃣ Install Crispy Form & crispy-bootstrap4:
- Use any of the following command
    ```
    pip install django-crispy-forms
    ```
    ```
    python -m pip install crispy-bootstrap4
    ```


# 2️⃣ `settings.py` of Project Folder

- **In the `INSTALLED_APPS` add:**
    ```
    crispy_forms
    ```
    ```
    crispy_bootstrap4
    ```
- **outside of `INSTALLED_APPS` add:**
    ```
    CRISPY_TEMPLATE_PACK = 'bootstrap4'
    ```

- **It will look like:**
    ```
    INSTALLED_APPS = [
        'django.contrib.admin',
        'django.contrib.auth',
        'django.contrib.contenttypes',
        'django.contrib.sessions',
        'django.contrib.messages',
        'django.contrib.staticfiles',
        
        'student_app',

        'crispy_forms',

        'crispy_bootstrap4',
    ]

    CRISPY_TEMPLATE_PACK = 'bootstrap4'
    CRISPY_ALLOWED_TEMPLATE_PACK = 'bootstrap4'
    ```



# 3️⃣ `Crispy form` use in project

### student_create.html
- For loading the form, add following tag on the top of the django template
```
{%load crispy_forms_tags %}
```

### student_create.html
- Now to style any form with django-crispy-forms-
    - replace `{{ form }}` with `{{ form|crispy }}`
